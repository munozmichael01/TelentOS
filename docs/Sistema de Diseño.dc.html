Superficies de IA / 05

La IA es una voz, no un lugar

Misma voz visual en todo el producto: tinta invertida = el agente habla (corto); papel claro = tú actúas. Máximo una superficie de agente visible por pantalla en reposo. Nunca se auto-aplica: se confirma o se descarta, con igual peso.

Regla nº1 · Anti-redundancia

La IA nunca repite lo que la pantalla ya muestra.
Añade lo que la pantalla no puede.

Comparación temporal · cruce de módulos · síntesis · extracción. Si un aviso ya es un flag visible (icono en la fila, contador, sección «sin datos») → se enlaza, no se repite.

Quién inicia
invocada · proactiva
EJE 02
Alcance

EJE 03
Naturaleza

EJE 04
Ciclo de vida
efímera · triage · conversacional

Regla de piel: hecho → superficie clara con filas accionables. juicio / generación → tinta oscura, texto corto. extracción → formulario claro pre-rellenado.

Procedencia · siempre visible · AgentBadge

Toda salida declara de dónde viene. La heurística nunca se oculta: si no hubo LLM, se dice.

Cómo aplicarlo a cualquier producto
1 · Importa las fuentes. 2 · Pega las variables CSS y el extend de Tailwind. 3 · Sustituye tu :root de shadcn por el mapeo de arriba — los componentes existentes (button, badge, card, table) se reskinean solos. 4 · Para acciones primarias y titulares, aplica shadow-hard + font-display con las recetas.
